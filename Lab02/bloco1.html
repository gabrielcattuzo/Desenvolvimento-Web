Introdução ao HTML5

HTML5 é a quinta versão do HTML, a linguagem padrão para criar páginas web.

Artigo: Novidades do HTML5

HTML5 trouxe várias novas tags e funcionalidades, incluindo suporte para vídeo, áudio, e gráficos 2D e 3D.

Sobre HTML5

HTML5 é mantido pela W3C e WHATWG, e se tornou o padrão recomendado para a construção de websites modernos.

Dicas de HTML5

Use a tag <header> para definir cabeçalhos de seções e páginas.

Organize a navegação do seu site com a tag <nav>.

Utilize <main> para agrupar o conteúdo principal da sua página.
